4 Rekomendasi Bahasa pemrograman

Penulis

Jika ingin terjun ke dunia IT, salah satu hal yang bisa kita lakukan adalah mempelajari bahasa pemrograman. Dengan mempelajari bahasa pemrograman, kita bisa melakukan banyak hal seperti membuat perangkat lunak, melakukan analisis data, bahkan merancang sistem operasi sekalipun. Maka berikut ini adalah rekomendasi bahasa pemrograman yang bisa kalian pelajari


Python

Python dikenal sebagai bahasa pemrograman yang cukup ramah untuk pemula. Bahasa ini menggunakan pendekatan object oriented programming namun juga bisa digunakan untuk pemrograman prosedural. Yang membuat Python dibilang ramah pemula adalah karena syntax bahasa pemrogramannya yang mendekati Bahasa Inggris. Phyton bisa digunakan untuk berbagai hal namun banyak digunakan untuk data science.


Javascript

Javascript adalah salah satu bahasa yang paling populer. Bahasa ini dikenal dikalangan front end, back end, hingga full stack developer. Javascript adalah sebuah bahasa pemrograman yang berjalan dalam sebuah laman web. Bahasa satu ini sering digunakan untuk memberikan interaktifitas pada website atau juga untuk menyimpan dan mengolah database


Java

Mungkin banyak mahasiswa jurusan IT yang mengenal bahasa pemrograman satu ini. Dikarenakan banyak universitas yang menggunakan bahasa ini untuk pembelajaran terutama di semester-semester awal. Java terkenal dengan konsep object oriented programmingnya dan memiliki syntax yang mirip dengan bahasa pemrograman keluarga C namun lebih mudah digarap untuk pemula. Java memiliki konsep yang terkenal yakni Write one, run anywhere. Maksud dari konsep ini adalah bahasa pemrograman Java bisa dijalankan dimana saja selama sebuah perangkat memiliki JVM atau Java Virtual Machine di dalamnya.


C++

Pengembangan dari bahasa C, bahsa pemrograman satu ini dikenal dengan tingkat kesulitannya dan tidak teralu direkomendasikan untuk pemrogram pemula. Namun tetap tidak ada salahnya mempelajari bahasa satu ini karena kekuatan dan fitur yang dimilikinya. C++ dikenal sebagai bahasa berorientasi obyek yang dekat dengan mesin.


Nah, itu dia rekomendasi bahasa pemrograman untuk para pemula. Bisa kalian pilih sesuai dengan kebutuhan kalian masing-masing.
